# 무드별 컬러·폰트 매칭 규칙

## 차분함 / Calm
- 색조: 저채도 블루·그린·그레이. 채도 낮게, 밝기 중상.
- 폰트: 얇은 세리프(Lora, Playfair Display) + 산세리프(Inter, DM Sans).
- 예: 명상, 수면, 웰니스 앱.

## 에너지 / Energetic
- 색조: 고채도 오렌지·레드·옐로우. 밝고 강렬하게.
- 폰트: 굵은 산세리프(Nunito, Poppins) + 굵은 디스플레이(Oswald).
- 예: 피트니스, 스포츠, 게임 앱.

## 프리미엄 / Premium
- 색조: 다크 네이비·차콜 배경에 골드·크림 액센트. 채도 낮게.
- 폰트: 엘레강트 세리프(Fraunces, Cormorant) + 얇은 산세리프(Inter).
- 예: 명품, 금융, 고급 서비스.

## 신뢰 / Trustworthy
- 색조: 미드 블루·네이비. 중채도, 중밝기.
- 폰트: 중간 웨이트 산세리프(Source Sans, Roboto).
- 예: 핀테크, B2B, 의료.

## 친근함 / Friendly
- 색조: 따뜻한 오렌지·코랄·라임. 중채도.
- 폰트: 둥근 산세리프(Nunito, Quicksand).
- 예: 소셜, 커뮤니티, 교육.

## 미니멀 / Minimal
- 색조: 거의 흑백. 액센트 한 가지만(매우 절제).
- 폰트: 극세 산세리프(Inter, Helvetica Neue).
- 예: 생산성, 노트, 포트폴리오.

## 자연 / Natural
- 색조: 어스톤—세이지그린·테라코타·베이지.
- 폰트: 유기적 세리프(Lora, Merriweather) + 편안한 산세리프.
- 예: 식품, 환경, 라이프스타일.
